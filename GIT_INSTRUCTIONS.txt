Решение: Выполните следующие команды в новом терминале:

1. Откройте новый терминал (PowerShell или Command Prompt)
2. Перейдите в папку проекта:
   cd "D:\Воркзилла\Netlify блог"

3. Добавьте изменения:
   git add blog.html

4. Сделайте коммит:
   git commit -m "Fix blog: add GitHub API integration + fallback articles"

5. Запушьте на GitHub:
   git push origin main

6. Netlify автоматически пересоберет сайт
